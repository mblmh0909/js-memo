#### （一）常用方法
##### String
	1.match 参数：字符串或正则表达式；返回：数组
	2.slice,subString,subStr;
		只有subStr第二个参数为长度，不支持副长度
		subString不支持负值自动转0，subStr长度当然不可能是负的也会自转0
